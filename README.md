# 67会员 VIP 购买页面

一个白色 Apple 风格的 VIP 购买网站，用于展示“购买一个 VIP 后还能继续购买更多 VIP”的趣味交互流程。页面支持 Go、Plus、Pro 三种会员类型，并会根据用户选择的 VIP 类型和购买数量自动计算最终价格。

## 项目特点

- 🌟 白色高级视觉风格
- 📱 同时适配手机和电脑
- 💳 支持 Go、Plus、Pro 三种 VIP 类型
- 🔢 根据会员类型和购买数量实时计算价格
- ✅ 点击购买后需要连续确认 10 次
- 💰 确认完成后显示对应的“已扣款”金额
- 🚀 纯 HTML / CSS / JavaScript 实现
- 🌐 适合部署到 GitHub Pages

## 价格规则

| VIP 类型 | 单价 | 计算方式 |
| --- | ---: | --- |
| Go | ¥6.70 / 个 | ¥6.70 × 数量 |
| Plus | ¥13.13 / 个 | ¥13.13 × 数量 |
| Pro | ¥91.78 / 个 | ¥91.78 × 数量 |

## 使用方式

1. 下载或克隆本项目。
2. 将 `index.html` 放到 GitHub 仓库根目录。
3. 在 GitHub 仓库中进入 `Settings` → `Pages`。
4. 选择部署分支并保存。
5. 等待 GitHub Pages 自动生成访问链接。

## 注意说明

本项目仅用于前端页面展示和交互演示，不包含真实支付功能，也不会产生真实扣款。页面中显示的“已扣款”金额只是根据选择的 VIP 类型和数量计算出的演示结果。

## 技术栈

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## 页面效果

用户可以选择不同 VIP 类型，调整购买数量，点击购买后进入 10 次确认流程。完成确认后，页面会显示最终计算出的扣款金额。
