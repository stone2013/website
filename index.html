新增功能说明

这个版本增加多语言、黑白主题和购物车。仍然是纯静态页面，适合直接部署到 GitHub Pages。

购物车价格怎么算？

页面使用整数“分”计算，避免小数误差。购物车总价是所有商品的单价乘以数量后相加。

Go¥6.70 × 数量
Plus¥13.13 × 数量
Pro¥91.78 × 数量
购物车总价全部商品金额相加

GitHub Pages 用法

1
上传 index.html

把 ZIP 解压后，将 index.html 上传到仓库根目录。

2
启用 Pages

Settings → Pages → Deploy from a branch → main → / root。

3
纯静态页面

不需要服务器、不需要数据库，也不需要真实支付接口。
