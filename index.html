GitHub Pages 用法

1
把文件名保持为 index.html

下载 ZIP 后，把里面的 index.html 上传到你的仓库根目录。

2
打开 Pages 发布

在仓库 Settings → Pages 里选择发布分支，保存后等待部署。

3
这是纯静态页面

页面只用 HTML、CSS、JavaScript，适合直接放到 GitHub Pages。
