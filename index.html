新增功能说明

这个版本加入会员卡生成器、订单记录系统、VIP 等级进化动画和支付成功烟花。仍然是纯静态页面，适合 GitHub Pages。

系统怎么计算？

页面使用整数“分”计算价格，订单成功后根据商品数量和可再买 VIP 数量生成经验值。

Go¥6.70 × 数量
Plus¥13.13 × 数量
Pro¥91.78 × 数量
经验值商品数量 × 12 + 可再买 VIP + 订单数 × 8

GitHub Pages 用法

1
上传 index.html

把 ZIP 解压后，将 index.html 上传到仓库根目录。

2
启用 Pages

Settings → Pages → Deploy from a branch → main → / root。

3
纯静态页面

不需要服务器、不需要数据库，也不需要真实支付接口。
